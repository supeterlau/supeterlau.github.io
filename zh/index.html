    启动两个 pg 节点 (using Docker Compose) 🔗(lr aka Logical Replication) mkdir lr mkdir -p pg1/{data,share} pg2/{data,share} touch docker-compose.yml content: start compose: docker-compose up 感谢 Debezium 的 postgresql-alpine-with-wal2json image 让安装 wal2json 很容易. 从容器中复制一份 pg_hba.conf.example 到 ./pg1/share/pg_hda.conf 和 ./pg2/share/pg_hda.conf start a new terminal: cd lr docker cp lr-pg1-1:/usr/local/share/postgresql/pg_hba.conf.sample lr/pg1/share/pg_hba.conf docker cp lr-pg1-1:/usr/local/share/postgresql/pg_hba.conf.sample
